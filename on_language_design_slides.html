int someMath(int x, int y) {
    assert(x >= 0, "Err message");
    assert(y >= 0, "Err message");
    assert(x + y >= x, "Err message");
    assert(x + y >= y, "Err message");
    return x + y;
}

The solution: Constraint type sets.
Adding additional information about the members into the set.
This could be done through a where clause or something similar.

fun g(x: i8) -> i8 where
    x < 10;
    x >= 0
{
    x + 5
